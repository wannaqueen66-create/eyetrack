**建议图题**

**建议图注正文**  
图4-1 展示了 QC 后样本中，不同 Experience 组在三类 AOI（table、window、equipment）上的注意分配占比（share\_pct）随 WWR 与 Complexity 的变化模式。纵轴为 trial 内总注视时长中分配给对应 AOI 的百分比，横轴为 WWR 水平（15、45、75）。蓝色与橙色线分别表示不同 Complexity 条件，阴影区域表示置信区间。整体上，table AOI 的注意分配占比高于 window 与 equipment；同时，Complexity 与 WWR 的变化会改变不同 AOI 的分配结构。

## 四、可直接放进正文的图表引用句式

### 图4-1 的引用句式
- 如图4-1所示，QC 后样本中不同 AOI 的注意分配占比呈现出明显的层级结构，其中 table AOI 整体上获得更高的分配比例。
- 图4-1进一步表明，Complexity 与 WWR 的变化会改变不同 AOI 的分配格局，而这种变化在不同 Experience 组之间存在一定程度的差异。

### 图4-2 的引用句式
- 从图4-2可以看出，基于注视次数（或注视率）的指标与 share\_pct 的总体方向基本一致，提示 AOI 分配差异不仅体现在停留时长，也体现在注视进入频率上。

### 图4-3 的引用句式
- 如图4-3所示，TTFF 的变化模式相较于 share\_pct 更不稳定，提示注意进入时延可能受个体差异与场景因素共同影响，因此应作为补充指标进行解释。

### 图4-4 的引用句式
- 图4-4从解释性角度进一步说明，Experience 与场景条件之间的交互并不是对所有 AOI 等比例生效，而更可能表现为不同 AOI 间分配结构的重组。

### 表4-2 的引用句式
- 由表4-2可见，Experience 主线 LMM 在多个 outcome 上存在稳定性限制，因此正文解释优先依据稳定性较高的模型，并辅以描述性与解释性图形结果。

### 表4-3 的引用句式
- 表4-3显示，在全样本轨道中，share\_logit 的主效应模型具有较好的稳定性，可作为正文结果的稳健性支撑。
